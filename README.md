# System Models
Mathematical modeling of various physical systems

## Physical Systems

- Mechanical (Mass-Spring-Damper)
- Electrical (RLC)
- Hydraulic (Tank Level)
- Thermal (Stove)
- Electronic (AmpOps)
- DC Converters (Buck, Boost...)
- DC Motor
- Suspended Pendulum
- Inverted Pendulum (Fixed or Mobile)
- Robotic Manipulator
- Mobile Robotics
